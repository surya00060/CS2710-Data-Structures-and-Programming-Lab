How to Execute:

1. Run "make" in terminal in the current directory where this folder is located
2. Then type "./dsu" to execute the code .

Usage:

1. Press 1 for Make Set Opeartion . Make Set Operation create one extra set and puts number from 10 .
2. Press 2 for Find Set Opeartion . Find Set Operation prints the leader set if found . 
3. Press 3 for Union Opeartion . Union Opeartion does union opeartion randomly to 2 sets .
4. Press 4 to exit .
